HTML CSS Exemplo

O que é HTML?

HTML cria a estrutura do site.

Tags comuns:

<h1> Título

<p> Parágrafo

<a> Link

<div> Divisão

O que é CSS?

CSS muda o visual do site.

Exemplo:

h1 { color: blue; }

.botao { background: green; }
